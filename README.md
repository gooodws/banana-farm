# Banana Farm

바나나 나무를 심고, 물 주고, 수확하고, 직접 껍질을 까서 희귀 바나나를 모으는 HTML/Canvas 게임입니다.

## 실행

`index.html`을 브라우저로 열면 바로 실행됩니다.

로컬 서버로 보고 싶으면 프로젝트 폴더에서 실행:

```powershell
python -m http.server 8792
```

그 다음 `http://127.0.0.1:8792/index.html`로 접속하면 됩니다.

## 조작

- PC: `WASD` 이동, `Space` 현재 칸 작업
- 모바일: 화면 하단 액션 버튼으로 농장 작업, 판매대/까기대는 직접 터치
- 농장: 갈기, 심기, 물 주기, 수확 순서로 진행
- 까기대: 수확한 바나나를 직접 까서 도감과 판매함을 채움
- 업그레이드: 별도 업그레이드 화면에서 자동화와 묘목/물/수확 관련 레벨 강화

## 파일 구조

- `index.html`: 게임 화면 구조
- `styles.css`: 반응형 UI와 모바일 터치 스타일
- `game.js`: 게임 로직, Canvas 렌더링, 저장 데이터

## 메모

게임 저장은 브라우저 `localStorage`를 사용합니다. 디자인 수정은 주로 `game.js`의 그리기 함수와 `styles.css`에서 하면 됩니다.
